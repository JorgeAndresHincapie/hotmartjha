# Catálogo de Productos – Hotmart

Este repositorio presenta un catálogo digital de productos disponibles en Hotmart, creado por **Jorge Andrés Hincapié Correa**.  
El objetivo es brindar una plataforma clara y organizada donde los usuarios puedan explorar cursos, ebooks y recursos motivacionales.

## 🌐 Publicación en GitHub Pages
Accede al catálogo en: https://JorgeAndresHincapie.github.io/catalogo-hotmart/

## 📂 Estructura del proyecto
- index.html → Página principal del catálogo
- assets/css → Estilos visuales
- assets/js → Scripts interactivos
- assets/images → Imágenes de productos
